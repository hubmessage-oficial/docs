## Conceituação

Método para envio do catálogo de produtos do WhatsApp Business. O destinatário recebe um card com link para visualizar o catálogo completo.

<Warning>
  Este recurso requer uma conta **WhatsApp Business** com catálogo e produtos configurados.
</Warning>

<Note>
  A estrutura de resposta é idêntica à da Z-API. Seu código existente que processa esses retornos continuará funcionando sem alterações.
</Note>
