## Conceituação

Método para envio de mensagens com carrossel de cards. Cada card pode conter imagem, texto e botões de ação (CALL, URL ou REPLY).

<Note>
  A estrutura de resposta é idêntica à da Z-API. Seu código existente que processa esses retornos continuará funcionando sem alterações.
</Note>
