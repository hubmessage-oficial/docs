## Conceituação

Envia um sticker (figurinha) para um contato. O campo `content.type` deve ser `STICKER` e o sticker é enviado via URL no array `attachments`.

<Note>
  Este endpoint só funciona quando a [janela de conversa](/conversation-window) de 24 horas estiver aberta.
</Note>
